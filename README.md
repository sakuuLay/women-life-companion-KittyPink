# Women Life Companion

Android app designed to support women’s daily life with calendar, to‑do list, mood and journal tracking, menstrual cycle management, SOS safety calling, routines, rewards, and a mini‑game. Built to simplify schedules, encourage wellness, and provide peace of mind.

---

## ✨ Features
- Calendar & To‑Do list
- Mood tracking with colors
- Short daily journal (200 chars)
- Menstrual cycle tracker
- SOS emergency calling
- Daily routines (wake, sleep, exercise, hobbies)
- Rewards system (points, streaks, badges)
- Mini‑game for daily reward

## 🛠 Tech Stack
- Kotlin (Android development)
- XML layouts (UI design)
- SQLite (local storage)
- Firebase (cloud sync & notifications)
- Material Design (UI guidelines)

## 🚀 Getting Started
1. Clone the repo: `git clone https://github.com/<your-username>/<repo-name>.git`
2. Open in Android Studio or VS Code
3. Build with Gradle
4. Run on an Android device (USB debugging)
